**Inputs:** 4 agent perspective docs in `docs/team-outputs/` + original Studio Reaktor study + Smeky FJORD HAMRY pitch + cabin_trae reference

## Goal

Produce a single-file HTML decision-aid deck for internal review (Martin + Smeky) that:
1. Diagnoses the gap between realized apartment quality and current public presentation (Muhu generic + old AirBnB photos)
2. Presents 7 naming options with SEO findings + honest pros/cons
3. Presents 8 tagline options with vibe categorization
4. Recommends 3 strongest combinations with rationale
5. Provides a decision framework (criteria for picking)
6. Shows execution roadmap (30/60/90 days)
7. Outlines negotiation points with Muhu/Pavel after Smeky decision

This deck is NOT the final brand. It is the input for Smeky meeting.

---

## Positioning (LOCKED)

**Headline:** Lokálně zakořeněný design v Jizerkách

**Brand promise (3 pillars):**
1. Architecturally rooted - Studio Reaktor "ingot" concept = autentický příběh, ne marketing
2. Quietly designed - Scandi-feeling interior bez transplant LARP-u
3. Mountain-honest - Jizerky bez nutnosti přidávat "Nordic" / "Fjord"

**Anti-positioning:**
- Reject FJORD HAMRY (cizopasný Nordic transplant, denies local heritage)
- Reject Muhu generic ("magic of Velké Hamry", "ancient mountain god" - lumps in with 100 similar listings)
- Embrace cabin_trae aesthetic - photo-led, minimal copy, authenticity > brand-heavy marketing

**Target persona:** 28-45 design-conscious couples, Praha/Brno/DE/PL, remote workers, "escape" mentality, follow design + coffee + Patagonia/Fjällräven aesthetic

**Anti-personas:** party crews, families >4, low-budget weekend hunters

- KOVÁRNA (Heavy heritage, less subtle)
